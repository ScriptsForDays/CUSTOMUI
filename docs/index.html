Dynamic Dropdown Updates

The :SetValues() method allows you to update dropdown values dynamically, which is perfect for scenarios where your exploit needs to refresh dropdown options based on runtime data.

Basic Usage

Lua 
local dropdown = Tab:Dropdown({
    Title = "Dynamic Dropdown",
    Desc = "Values can be updated dynamically",
    Values = {
        { Title = "Option 1", Icon = "file" },
        { Title = "Option 2", Icon = "folder" },
        { Title = "Option 3", Icon = "settings" },
    },
    Value = "Option 1",
    Callback = function(option)
        print("Selected:", option.Title)
    end
})

-- Later, update the dropdown values
dropdown:SetValues({
    { Title = "New Option A", Icon = "star" },
    { Title = "New Option B", Icon = "heart" },
    { Title = "New Option C", Icon = "zap" },
})

Preserving Selection

The method attempts to preserve the current selection by matching titles. If the current selection is not in the new values, it will be cleared.

Custom Accent Customization

Customize UI accents without creating a full theme. These overrides persist when switching themes, giving you fine-grained control over the UI appearance.

Available Methods

WindUI:SetCustomAccent(color)

Set the accent color

WindUI:SetCustomBackground(color)

Set the background color (supports gradients)

WindUI:SetCustomText(color)

Set the text color

WindUI:SetCustomButton(color)

Set the button color (supports gradients)

WindUI:SetCustomIcon(color)

Set the icon color

WindUI:SetCustomDialog(color)

Set the dialog background color

WindUI:SetCustomOutline(color)

Set the outline color

WindUI:SetCustomProperty(name, value)

Set any custom theme property

Basic Examples

Solid Colors 
-- Set custom accent color
WindUI:SetCustomAccent(Color3.fromHex("#30ff6a"))

-- Set custom background
WindUI:SetCustomBackground(Color3.fromHex("#1a1a1a"))

-- Set custom text color
WindUI:SetCustomText(Color3.fromHex("#ffffff"))

Gradient Examples

Gradients 
-- Custom background with gradient
WindUI:SetCustomBackground(WindUI:Gradient({
    ["0"] = { Color = Color3.fromHex("#0f0c29"), Transparency = 0 },
    ["100"] = { Color = Color3.fromHex("#302b63"), Transparency = 0.1 },
}, {
    Rotation = 45,
}))

-- Custom button with gradient
WindUI:SetCustomButton(WindUI:Gradient({
    ["0"] = { Color = Color3.fromHex("#30FF6A"), Transparency = 0 },
    ["100"] = { Color = Color3.fromHex("#e7ff2f"), Transparency = 0 },
}, {
    Rotation = 60,
}))

Managing Overrides

Clear Overrides 
-- Get all current custom overrides
local overrides = WindUI:GetCustomOverrides()
print("Current overrides:", overrides)

-- Clear a specific custom property
WindUI:ClearCustomProperty("Accent")

-- Clear all custom overrides
WindUI:ClearCustomOverrides()

Advanced Usage

Generic Property Setter 
-- Set any custom theme property
WindUI:SetCustomProperty("WindowBackground", Color3.fromHex("#0f172a"))
WindUI:SetCustomProperty("TabBackground", Color3.fromHex("#1e293b"))
WindUI:SetCustomProperty("ElementBackground", Color3.fromHex("#1e293b"))

Complete Example

Here's a complete example showing both customizations in action:

Complete Example 
-- Load WindUI
local WindUI = loadstring(game:HttpGet('https://raw.githubusercontent.com/ScriptsForDays/CUSTOMUI/refs/heads/main/dist/main.lua'))()

-- Set custom accent
WindUI:SetCustomAccent(Color3.fromHex("#30ff6a"))

-- Create window
local Window = WindUI:Window({
    Title = "My Custom UI",
    Size = { 500, 400 }
})

local Tab = Window:Tab({
    Title = "Main",
    Icon = "home"
})

-- Create dynamic dropdown
local dropdown = Tab:Dropdown({
    Title = "Dynamic Dropdown",
    Values = {
        { Title = "Option 1" },
        { Title = "Option 2" },
    },
    Callback = function(option)
        print("Selected:", option.Title)
    end
})

-- Update dropdown later
task.wait(5)
dropdown:SetValues({
    { Title = "New Option A" },
    { Title = "New Option B" },
    { Title = "New Option C" },
})
