Installation

Get started with WindUI Custom

Quick Start

Load WindUI Custom directly from GitHub:

local WindUI = loadstring(game:HttpGet('https://raw.githubusercontent.com/ScriptsForDays/CUSTOMUI/refs/heads/main/dist/main.lua'))()

Basic Usage

After loading WindUI, create a window and start adding elements:

-- Load WindUI
local WindUI = loadstring(game:HttpGet('https://raw.githubusercontent.com/ScriptsForDays/CUSTOMUI/refs/heads/main/dist/main.lua'))()

-- Create a window
local Window = WindUI:CreateWindow({
    Title = "My Script",
    Author = "by YourName",
    Folder = "myscript",
    Icon = "settings"
})

-- Create a tab
local Tab = Window:Tab({
    Title = "Main",
    Icon = "home"
})

-- Add a button
Tab:Button({
    Title = "Click Me",
    Icon = "mouse-pointer-click",
    Callback = function()
        print("Button clicked!")
    end
})

Key System

WindUI includes a built-in key system to protect your scripts. You can add it when creating a window by including the KeySystem parameter in your configuration.

Basic Key System

The simplest way to add a key system is with a single key or multiple keys:

-- Single key
local Window = WindUI:CreateWindow({
    Title = "My Script",
    Author = "by YourName",
    Folder = "myscript",
    Icon = "settings",
    KeySystem = {
        Key = "YOUR-KEY-HERE",
        SaveKey = true, -- Save key locally (optional)
        Note = "Enter your key to access the script" -- Optional note text
    }
})

-- Multiple keys (array)
local Window = WindUI:CreateWindow({
    Title = "My Script",
    Author = "by YourName",
    Folder = "myscript",
    Icon = "settings",
    KeySystem = {
        Key = {"KEY-1", "KEY-2", "KEY-3"}, -- Multiple valid keys
        SaveKey = true
    }
})

Key System with URL

You can add a URL where users can get their key (Discord, Linkvertise, Pastebin, etc.):

local Window = WindUI:CreateWindow({
    Title = "My Script",
    Author = "by YourName",
    Folder = "myscript",
    Icon = "settings",
    KeySystem = {
        Key = "YOUR-KEY-HERE",
        URL = "https://discord.gg/yourserver", -- Link to get key
        SaveKey = true,
        Note = "Join our Discord to get your key!"
    }
})

Key System with Discord Button

Add a Discord button next to the Exit button that copies your Discord link to clipboard:

-- Simple: Just provide the Discord URL string
local Window = WindUI:CreateWindow({
    Title = "My Script",
    Author = "by YourName",
    Folder = "myscript",
    Icon = "settings",
    KeySystem = {
        Key = "YOUR-KEY-HERE",
        Discord = "https://discord.gg/yourserver", -- Discord button enabled with this URL
        SaveKey = true
    }
})

-- Advanced: Control with table
local Window = WindUI:CreateWindow({
    Title = "My Script",
    Author = "by YourName",
    Folder = "myscript",
    Icon = "settings",
    KeySystem = {
        Key = "YOUR-KEY-HERE",
        Discord = {
            Enabled = true, -- Enable/disable Discord button
            URL = "https://discord.gg/yourserver" -- Discord invite link
        },
        SaveKey = true
    }
})

Note: The Discord button appears next to the Exit button with a Discord icon on the left. When clicked, it copies the Discord URL to clipboard and shows a notification.

Key System with Thumbnail

Add a custom thumbnail image to the key system dialog:

local Window = WindUI:CreateWindow({
    Title = "My Script",
    Author = "by YourName",
    Folder = "myscript",
    Icon = "settings",
    KeySystem = {
        Key = "YOUR-KEY-HERE",
        Thumbnail = {
            Image = "rbxassetid://123456789", -- Roblox asset ID or image URL
            Title = "My Script", -- Optional title
            Width = 200 -- Optional width (default: 200)
        },
        SaveKey = true
    }
})

API-Based Key System

WindUI supports integration with key system services like PlatoBoost, Panda Development, and Luarmor:

local Window = WindUI:CreateWindow({
    Title = "My Script",
    Author = "by YourName",
    Folder = "myscript",
    Icon = "settings",
    KeySystem = {
        Note = "Enter your key to access the script",
        SaveKey = true,
        API = {
            -- PlatoBoost
            {
                Type = "platoboost",
                ServiceId = 1234,
                Secret = "your-secret-key"
            },
            -- Panda Development
            {
                Type = "pandadevelopment",
                ServiceId = "your-service-id"
            },
            -- Luarmor
            {
                Type = "luarmor",
                ServiceId = "your-service-id"
            }
        }
    }
})

Key System Parameters

Parameter Type Description
Key string/table Valid key(s). Can be a single string or table of strings for multiple keys.
SaveKey boolean Automatically save and load the key locally (default: false). If true, users won't need to enter the key again.
Note string Optional note text displayed in the key system dialog.
URL string Link where users can get their key (Discord, Linkvertise, Pastebin, etc.).
Thumbnail table Custom thumbnail image. Contains Image (string), Title (string, optional), and Width (number, optional, default: 200).
API table Array of API service configurations. Supported types: "platoboost", "pandadevelopment", "luarmor".
Discord string/table Discord button configuration. Can be a string (Discord URL) or table with Enabled (boolean) and URL (string). When enabled, adds a Discord button next to the Exit button that copies the Discord link to clipboard.

How It Works

  • When KeySystem is provided, the window will not load until a valid key is entered
  • If SaveKey is true, the key is saved locally and automatically loaded on next run
  • If API is provided, keys are validated through the specified service(s)
  • If URL is provided, users can click to open the link and get their key
  • The key system dialog appears before the main window, blocking access until a valid key is entered

Requirements

  • Roblox exploit/executor with HTTP support
  • Internet connection to load from GitHub
