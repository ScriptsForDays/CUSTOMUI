# WindUI Custom Documentation

This directory contains the GitHub Pages documentation for WindUI Custom.

## Local Development

To preview the documentation locally:

1. Install a local server (Python, Node.js, etc.)
2. Navigate to the `docs` directory
3. Start a local server:
   - Python: `python -m http.server 8000`
   - Node.js: `npx http-server -p 8000`
4. Open `http://localhost:8000` in your browser

## Deployment

The documentation is automatically deployed to GitHub Pages when changes are pushed to the `main` branch.

Access the live site at: `https://ScriptsForDays.github.io/CUSTOMUI/`
