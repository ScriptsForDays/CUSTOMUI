### 1. Update Repository Description on GitHub

To add the documentation link to your repository description (like the original WindUI repo):

1. Go to: https://github.com/ScriptsForDays/CUSTOMUI
2. Click the gear icon (‚öôÔ∏è) next to "About" section

4. Optionally update the description to: `Enhanced Roblox UI Library with Dynamic Updates & Custom Accent Customization`
5. Click "Save changes"

### 2. Enable GitHub Pages

1. Go to: https://github.com/ScriptsForDays/CUSTOMUI/settings/pages
2. Under "Source", select: **"GitHub Actions"**
3. The site will automatically deploy when you push changes

### 3. Push Your Changes

After authentication, push all changes:
```powershell
git push origin main
```
