API Reference

WindUI:CreateWindow

Creates a new window container for your UI.

Parameters

Parameter Type Description
Title string Window title displayed in the topbar
Author string Author name displayed below title
Folder string Folder name for saving configs
Icon string Icon name for the window
Size UDim2 Optional custom window size
NewElements boolean Use new element styling (default: false)
HideSearchBar boolean Hide the search bar (default: false)
OpenButton table Configuration for the open button

Example

local Window = WindUI:CreateWindow({
    Title = "My Script",
    Author = "by Developer",
    Folder = "myscript",
    Icon = "settings",
    NewElements = true
})

Methods

Window:Tab(config)

Creates a new tab in the window.

Window:Section(config)

Creates a section directly in the window.

Window:Tag(config)

Adds a tag to the window topbar.

Window:Destroy()

Destroys the window and all its elements.

Tab:Button

Creates a clickable button element.

Parameters

Parameter Type Description
Title string Button text

Locks the button, preventing clicks.

Button:Unlock()

Unlocks the button, allowing clicks.

Tab:Dropdown

Creates a dropdown menu for selecting options.

Parameters

Parameter Type Description
Title string Dropdown label
Desc string Optional description
Values table Array of option tables: {Title, Desc?, Icon?, Callback?}
Value string/table Initial selection (string for single, table for multi)
Multi boolean Enable multi-select (default: false)
AllowNone boolean Allow deselecting all options (default: false)
SearchBarEnabled boolean Enable search functionality (default: false)
MenuWidth number Custom menu width
Locked boolean Disable the dropdown (default: false)
Callback function Function called when selection changes: function(option)

Methods

Dropdown:SetValues(newValues)

Custom Enhancement: Updates dropdown options dynamically. If newValues is nil and a DataSource exists, it will refresh from the original source. Preserves selection when possible.

Dropdown:RefreshData()

Custom Enhancement: Refetches data from the original DataSource (function or ModuleScript). Returns true if successful.

Dropdown:LockValues(values)

Locks specific values from being selected.

Dynamic Data Sources

Custom Enhancement: The Values parameter can be:

  • Table: Static list of options (default behavior)
  • Function: Function that returns a table of options - called automatically
  • String: ModuleScript name/path - searches game and requires the module

When using a function or ModuleScript, the dropdown automatically fetches data on creation and can be refreshed with :RefreshData().

Slider:SetMax(newMax)

Updates the maximum value.

Slider:SetMin(newMin)

Updates the minimum value.

WindUI Custom Methods

Custom enhancement methods for theme customization.

Custom Accent Methods

WindUI:SetCustomAccent(color)

Sets a custom accent color. Accepts Color3 or Gradient.

WindUI:SetCustomBackground(color)

Sets a custom background color. Accepts Color3 or Gradient.

WindUI:SetCustomText(color)

Sets a custom text color.

WindUI:SetCustomButton(color)

Sets a custom button color. Accepts Color3 or Gradient.

WindUI:SetCustomIcon(color)

Sets a custom icon color.

WindUI:SetCustomDialog(color)

Sets a custom dialog background color.

WindUI:SetCustomOutline(color)

Sets a custom outline color.

WindUI:SetCustomProperty(name, value)

Sets any custom theme property by name.

WindUI:ClearCustomProperty(name)

Clears a specific custom property.

WindUI:ClearCustomOverrides()

Clears all custom property overrides.

WindUI:GetCustomOverrides()

Returns a table of all current custom overrides.
