Quick Start

Load WindUI Custom and create your first window:

-- Load WindUI
local WindUI = loadstring(game:HttpGet('https://raw.githubusercontent.com/ScriptsForDays/CUSTOMUI/refs/heads/main/dist/main.lua'))()

-- Create a window
local Window = WindUI:CreateWindow({
    Title = "My Script",
    Author = "by YourName",
    Folder = "myscript",
    Icon = "settings"
})

-- Create a tab
local Tab = Window:Tab({
    Title = "Main",
    Icon = "home"
})

-- Add a button
Tab:Button({
    Title = "Click Me",
    Icon = "mouse-pointer-click",
    Callback = function()
        print("Button clicked!")
    end
})

Key Features

  • Custom Theme Customization: Override any theme property with methods like SetCustomAccent(), SetCustomDropdownSelected(), etc.
  • Automatic Config Saving: All custom theme colors are automatically saved and loaded with configs
  • Named Configs: Users can create multiple named configs and load them by typing the name
  • Dynamic Updates: UI updates immediately when you change colors or settings
  • Gradient Support: Use gradients for backgrounds, buttons, and other elements
  • Key System: Built-in key system with Discord integration and API support
  • ThemeProperty Feature: Colorpickers automatically display current UI colors
